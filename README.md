📧 Smart Unified Email Aggregator

A real-time email management system that integrates multiple email providers like Gmail, Outlook, and Zoho into a single unified dashboard.

🚀 Features

* 🔗 Multi-provider email integration (Gmail, Outlook, Zoho)
* 📥 Unified inbox view
* 🧠 Machine Learning-based spam detection (TF-IDF + Logistic Regression)
* ⭐ Priority highlighting (High, Medium, Low)
* 🗂️ Email categorization (Important, Promotional, Personal)
* 📄 Email summarization
* 🔊 Voice-based email reading
* ⏰ Reminder alerts

🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* IMAP Protocol
* Pandas, Joblib

📂 Project Structure

/frontend
/backend
/models
/docs

⚙️ How to Run

```bash
pip install -r requirements.txt
streamlit run app1.py
```
