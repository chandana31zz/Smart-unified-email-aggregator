# 📧 Smart Unified Email Aggregator

A real-time email management system that integrates multiple email providers like Gmail, Outlook, and Zoho into a single unified dashboard.

---

## 🚀 Features

* 🔗 Multi-email integration (Gmail, Outlook, Zoho)
* 📥 Unified inbox view
* 🧠 ML-based spam detection (TF-IDF + Logistic Regression)
* ⭐ Priority highlighting (High, Medium, Low)
* 🗂️ Email categorization (Important, Promotional, Personal)
* 📄 Email summarization
* 🔊 Voice-based email reading
* ⏰ Reminder alerts for important emails

---

## 🛠️ Tech Stack

* **Frontend/UI:** Streamlit
* **Backend:** Python
* **Machine Learning:** Scikit-learn
* **Email Protocol:** IMAP
* **Libraries:** Pandas, Joblib, NLTK

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/smart-unified-email-aggregator.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the application:

```
streamlit run app.py
```
